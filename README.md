## 🛠 Cài Đặt
Hướng dẫn cài đặt dự án.

```bash
# Clone repo
git clone: https://github.com/ThanhHienIT2004/Job-Search-Application
cd your-project

# Cài đặt dependencies
npm install

# Chạy dự án
npm start
```
